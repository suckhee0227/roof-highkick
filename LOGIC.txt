=====================================
4. 지붕 뚫고 하이킥! - 게임 로직 설명서
(4-1, 4-2, 4-3 기획서 기반)
=====================================

[게임 개요]
- 두 팀이 동시에 수학 문제를 풀고, 먼저 맞춘 팀이 테트리스 조각을 배치
- 4x7 그리드를 아래부터 채워 지붕 높이에 먼저 도달하면 승리

=====================================
화면 4-1: 숫자 선택 (카드 뒤집기)
=====================================

[흐름]
1. 양팀의 목표 숫자 카드가 뒤집혀 있음
2. 각 팀이 자신의 카드를 터치하여 뒤집음
3. 뒤집을 때 '뻥' 효과음
4. 양팀 모두 뒤집으면 화면 4-2로 이동

[목표 숫자 범위]
- 양수: +20 ~ +60 (70% 확률)
- 음수: -10 ~ -20 (30% 확률)
- 각 팀은 서로 다른 목표 숫자를 받음

[화면 구성]
- 왼쪽: 팀 1 카드 (라운드 번호 뱃지 표시)
- 오른쪽: 팀 2 카드
- 뒤집기 전: "?" 표시
- 뒤집은 후: "목표 숫자" + 숫자 표시

=====================================
화면 4-2: 숫자 더하거나 빼서 맞추기
=====================================

[핵심 메카닉: 누적 계산]
- 현재 값은 0에서 시작
- 부호(+/-) 선택 후 숫자(1~9) 하나를 누르면 즉시 연산
- 한 번에 하나의 숫자만 적용 (다중 자릿수 불가)
- 예시:
  '+' 누르고 5 -> 0+5 = 5
  '+' 누르고 3 -> 5+3 = 8
  '-' 누르고 2 -> 8-2 = 6

[동시 플레이]
- 양팀이 같은 화면에서 동시에 플레이
- 각 팀은 자신의 계산기만 조작
- 먼저 목표 숫자에 도달한 팀이 승리
- 승리 팀만 테트리스(화면 4-3)로 이동

[부호 선택 규칙]
- 기본값은 '+'
- '+' 또는 '-'를 누르면 해당 부호가 선택됨
- 현재 선택된 부호로 다음 숫자 연산이 적용됨

[숫자 입력 규칙]
- 숫자는 1~9 중 하나만 선택 가능
- 부호 1회 + 숫자 1회 = 1회 연산
- 만약 숫자 '1'을 누르고 다시 '2'를 누르면, 최종으로 '2'의 값 사용

[화면 구성]
- 좌측: 팀 1 (목표 카드 + 현재값 + 부호 버튼 + 숫자 패드)
- 우측: 팀 2 (동일 구성)
- 각 요소에 번호 뱃지 표시

=====================================
화면 4-3: 테트리스 블록 쌓기
=====================================

[승리 팀만 표시]
- 수학에서 이긴 팀의 화면만 나타남
- 3개의 조각을 배치한 후 화면 4-1로 돌아가 재경기

[조각 선택]
- 1칸~5칸(펜토미노) 범위의 블록
- 최소 10개 이상에서 선택 가능
- 항상 배치 가능한 조각만 생성

[블록 회전]
- 회전 버튼으로 90도 시계방향 회전
- 회전 후에도 배치 가능 여부 자동 확인

[배치 규칙]
- 4방향 화살표로 위치 이동
- 겹침 불가 (기존 블록과 겹치는 위치 자동 건너뜀)
- 접지 필수 (바닥 또는 기존 블록 위에만 배치)
- 배치 버튼으로 확정

[배치 후]
- 3개 배치 완료 -> 다음 라운드 (화면 4-1로)
- 조각 풀 자동 보충

=====================================
승리 조건
=====================================

[높이 계산]
- 아래(row 0)부터 연속으로 완전히 채워진 행만 높이로 인정
- 4칸 모두 채워져야 해당 행이 완성됨
- 빈칸이 하나라도 있으면 그 위는 높이로 인정 안 됨

[승리]
- 7행 전부 완전히 채우면 승리 (4x7 = 28칸)

=====================================
전체 게임 흐름
=====================================

R1: 카드뒤집기 -> 동시 수학 -> 이긴팀 테트리스 ->
R2: 카드뒤집기 -> 동시 수학 -> 이긴팀 테트리스 ->
R3: ...
(지붕 도달 시 게임 종료)

- 조각 배치: E4
- 승리: C5-E5-G5-C6 팡파레
